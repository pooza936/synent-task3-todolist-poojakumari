# 📝 Todo List App

A simple command-line **Todo List** built with Python. Save, complete, and delete tasks — all stored in a JSON file so your data is never lost!

---

## 🎯 Features

- ✅ Add new tasks
- 📋 View all tasks with status
- ✔️ Mark tasks as complete
- 🗑️ Delete tasks
- 💾 Auto-saves to `todos.json`
- 🕐 Shows date/time each task was created

---

## 🛠️ Requirements

- Python 3.6 or higher
- No extra libraries needed (uses built-in `json`, `os`, `datetime`)

---

## 🚀 How to Run

1. **Clone the repo:**
   ```bash
   git clone https://github.com/YOUR_USERNAME/todo-app.git
   cd todo-app
   ```

2. **Run the app:**
   ```bash
   python todo.py
   ```

3. **Follow the menu:**
   ```
   1. Add Task
   2. View Tasks
   3. Complete Task
   4. Delete Task
   5. Exit
   ```
